# Minesweeper Game

Minesweeper is a classic single-player puzzle game. This project is a Java implementation of Minesweeper with a graphical user interface (GUI).

## Features

- Three difficulty levels: Beginner, Intermediate, and Advanced.
- Timer to keep track of the time played.
- Explosion animation and sound when a mine is clicked.
- Right-click to mark cells as mines.
- Automatic uncovering of adjacent cells with a second click on revealed cells with the correct number of adjacent mines.
